# 🏦 ATM Management System (Java)

A simple ATM simulation project built using Java.
This project includes console-based ATM, advanced file storage system, and a GUI version using Java Swing.

---

## 🚀 Features

### 🔹 Console ATM
- PIN authentication
- Balance checking
- Deposit & Withdraw
- Transaction history
- File storage (history saved in history.txt)

### 🔹 Advanced Features
- 3 PIN attempt limit
- Transaction history stored permanently
- .gitignore setup (professional structure)

### 🔹 GUI ATM (Java Swing)
- Window-based interface
- Deposit & Withdraw buttons
- Balance checking
- Simple user-friendly design

---

## 🛠 Technologies Used

- Java (JDK 25)
- Java Swing (GUI)
- Git & GitHub
- File Handling (FileWriter & BufferedReader)
